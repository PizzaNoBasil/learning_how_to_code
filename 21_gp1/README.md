## Chapter III

–ö–æ–º–∞–Ω–¥–Ω–æ–µ –∑–∞–¥–∞–Ω–∏–µ –Ω–∞—Ü–µ–ª–µ–Ω–æ –Ω–∞ –∑–Ω–∞–∫–æ–º—Å—Ç–≤–æ —Å —Ñ–∞–π–ª–∞–º–∏. 

–§–∞–π–ª ‚Äî —ç—Ç–æ –Ω–∞–±–æ—Ä –¥–∞–Ω–Ω—ã—Ö, —Å–æ—Ö—Ä–∞–Ω–µ–Ω–Ω—ã–π –Ω–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–µ –≤ –≤–∏–¥–µ –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ –±–∏—Ç–æ–≤. –í Python —Å—É—â–µ—Å—Ç–≤—É–µ—Ç –¥–≤–∞ —Ç–∏–ø–∞ —Ñ–∞–π–ª–æ–≤:
- —Ç–µ–∫—Å—Ç–æ–≤—ã–µ
- –±–∏–Ω–∞—Ä–Ω—ã–µ

–¢–µ–∫—Å—Ç–æ–≤—ã–µ —Ñ–∞–π–ª—ã ‚Äî —ç—Ç–æ —Ñ–∞–π–ª—ã —Å —Å–æ–¥–µ—Ä–∂–∏–º—ã–º, –ø–æ–Ω—è—Ç–Ω—ã–º –¥–ª—è —á–µ–ª–æ–≤–µ–∫–∞. –í –Ω–∏—Ö —Ö—Ä–∞–Ω—è—Ç—Å—è –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–∏ —Å–∏–º–≤–æ–ª–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ —á–µ–ª–æ–≤–µ–∫ –ø–æ–Ω–∏–º–∞–µ—Ç. –í –±–∏–Ω–∞—Ä–Ω—ã—Ö —Ñ–∞–π–ª–∞—Ö –¥–∞–Ω–Ω—ã–µ –æ—Ç–æ–±—Ä–∞–∂–∞—é—Ç—Å—è –≤ –∑–∞–∫–æ–¥–∏—Ä–æ–≤–∞–Ω–Ω–æ–π —Ñ–æ—Ä–º–µ: —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º –Ω—É–ª–µ–π –∏ –µ–¥–∏–Ω–∏—Ü –≤–º–µ—Å—Ç–æ –ø—Ä–æ—Å—Ç—ã—Ö —Å–∏–º–≤–æ–ª–æ–≤. 

–õ—é–±—É—é –æ–ø–µ—Ä–∞—Ü–∏—é —Å —Ñ–∞–π–ª–æ–º –º–æ–∂–Ω–æ —Ä–∞–∑–±–∏—Ç—å –Ω–∞ —Ç—Ä–∏ –∫—Ä—É–ø–Ω—ã—Ö —ç—Ç–∞–ø–∞:
- –æ—Ç–∫—Ä—ã—Ç–∏–µ —Ñ–∞–π–ª–∞
- –≤—ã–ø–æ–ª–Ω–µ–Ω–∏–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ (—á—Ç–µ–Ω–∏–µ –∏–ª–∏ –∑–∞–ø–∏—Å—å)
- –∑–∞–∫—Ä—ã—Ç–∏–µ —Ñ–∞–π–ª–∞
–í Python –µ—Å—Ç—å –≤—Å—Ç—Ä–æ–µ–Ω–Ω–∞—è —Ñ—É–Ω–∫—Ü–∏—è open(). –° –µ–µ –ø–æ–º–æ—â—å—é –º–æ–∂–Ω–æ –æ—Ç–∫—Ä—ã—Ç—å –ª—é–±–æ–π —Ñ–∞–π–ª –Ω–∞ –∫–æ–º–ø—å—é—Ç–µ—Ä–µ. 

–î–µ–ª–∞–µ—Ç—Å—è —ç—Ç–æ —Å–ª–µ–¥—É—é—â–∏–º –æ–±—Ä–∞–∑–æ–º:

–ö–∞–∫ –±—ã —Å–æ–∑–¥–∞–µ—Ç—Å—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è —Ñ–∞–π–ª–∞, —Å –∫–æ—Ç–æ—Ä–æ–π –≤ –¥–∞–ª—å–Ω–µ–π—à–µ–º –º—ã —Å–º–æ–∂–µ–º —Ä–∞–±–æ—Ç–∞—Ç—å.

–û—Å–Ω–æ–≤–Ω—ã–µ —Ä–µ–∂–∏–º—ã: —á—Ç–µ–Ω–∏–µ (r) –∏ –∑–∞–ø–∏—Å—å (w). –û –¥—Ä—É–≥–∏—Ö —Ä–µ–∂–∏–º–∞—Ö —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ–º –ø–æ—á–∏—Ç–∞—Ç—å —Å–∞–º–æ—Å—Ç–æ—è—Ç–µ–ª—å–Ω–æ. –ü–æ —É–º–æ–ª—á–∞–Ω–∏—é –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è —Ä–µ–∂–∏–º —á—Ç–µ–Ω–∏—è (r), –µ—Å–ª–∏ –¥—Ä—É–≥–æ–µ –Ω–µ —É–∫–∞–∑–∞–Ω–æ.
